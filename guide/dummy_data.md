# Dummy Data untuk Performance Chart

Halo! Di bawah ini adalah data simulasi (*dummy data*) rekap harian selama **40 hari terakhir**, khusus disiapkan untuk menguji tampilan `PerformanceChart.tsx` Anda. Data ini mencakup metrik *mined* dan *burned*.

### Instruksi Penggunaan:
1. Buka **Supabase Dashboard** Anda.
2. Masuk ke menu **SQL Editor**.
3. *Copy* (Salin) *script* SQL di dalam kotak hitam di bawah ini.
4. *Paste* (Tempel) ke dalam SQL Editor lalu jalankan (*Run*).

> **Catatan:** Script ini aman untuk dijalankan berkali-kali karena menggunakan sistem `UPSERT` (hanya meng-*update* jika ada tanggal yang sama).

---
